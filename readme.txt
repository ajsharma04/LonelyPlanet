About the app

The home page would be the first taxonomy node, which in this case is world.html.

The navigation on the right would display the child destinations and the top nav will have a link to parent destination.

The structure of file generated is to ease any maintenance.


****************************

Steps to run the application

Running from Windows

1. Unzip the given project folder
2. Go to the project directory. Open the solution, build.
3. Open CMD
4. Navigate to project directory->LPMain->bin->debug
5. From CMD run LPMain.exe with 3 params
	i) Destination xml path - eg "C:\Users\ajay\Downloads\lonely_planet_coding_exercise\lonely_planet_coding_exercise\destinations.xml"
	ii) Taxonomy xml path - eg "C:\Users\ajay\Downloads\lonely_planet_coding_exercise\lonely_planet_coding_exercise\taxonomy.xml"
	iii) Output Dir path - eg "C:\Users\ajay\Documents\LonelyPlanet\"
6. Run
